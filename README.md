# Bug Hunt

A cyberpunk pixel art browser game built with Phaser 4, Vite, and TypeScript.

Hunt glitchy pixel bugs across a neon-lit corrupted cityscape. Inspired by Duck Hunt.

## Stack
- [Phaser 4](https://phaser.io)
- [Vite](https://vitejs.dev)
- TypeScript

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
